# API de Usuários com FastAPI

Projeto simples de API CRUD desenvolvido com Python e FastAPI.

## Tecnologias utilizadas

- Python
- FastAPI
- SQLite

## Funcionalidades

- Criar usuário
- Listar usuários
- Atualizar usuário
- Deletar usuário

## Como executar

1. Instale as dependências

pip install fastapi uvicorn

2. Execute o servidor

uvicorn main:app --reload

3. Abra no navegador

http://127.0.0.1:8000/docs
